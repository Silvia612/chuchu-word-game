# 单词打卡小助手

## 快速开始

### 1. 上传单词作业
把老师布置的单词作业拍照，然后对我说：
```
上传单词作业，图片路径：[图片路径]
```

我会自动识别图片中的单词并保存。

### 2. 每晚学习
每天晚上9点，我会自动提醒你开始学习。你也可以随时对我说：
```
开始学习今天的单词
```

### 3. 游戏化学习模式
- **听写模式**：我读单词，你拼写
- **选择题**：看中文选英文
- **填空题**：补全单词

### 4. 生成打卡报告
学习完成后，对我说：
```
生成打卡报告
```

我会生成一张精美的打卡图片，你可以转发给老师。

## 设置信息

告诉我你的信息：
```
我的名字是[你的名字]
老师叫[老师名字]
```

## 学习统计

随时查看你的学习数据：
```
查看学习统计
```

## 数据存储

所有数据保存在：
- 单词数据：`data.json`
- 学习记录：`history/`
- 打卡报告：`reports/`
